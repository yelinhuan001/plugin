========================================================
  巨魔通用工具箱 v2.0  — 悬浮球 / 会员 / 去广告
========================================================

【功能】
  1. 悬浮球界面（可拖动，点「魔」打开面板）
  2. 针对注入的 App：扫描会员相关本地数据
  3. 启发式「强制会员」（UserDefaults + isVip 等方法 Hook）
  4. 启发式去广告（隐藏广告视图 + 拦截常见广告 SDK）

【重要限制 — 请先读】
  · 必须用 TrollFools 把 dylib 注入到「每个」目标 App
    （不是系统级全局，巨魔注入是按 App 的）
  · 会员：只对「纯本地判断」的简单 App 可能有效
    服务端校验 / IAP 收据 / 加密登录态 → 无法本地改成真会员
  · 去广告：不能保证 100% 去掉所有广告联盟
  · 仅建议用于自己的设备学习调试，勿用于盗版传播

【编译（无 Mac 用 GitHub Actions）】
  见：无Mac-直接下载dylib.md
  上传本工程 → Actions 变绿 → 下载 TrollDylibPlugin.dylib

【注入】
  1. TrollFools 选择 App → 注入 TrollDylibPlugin.dylib
  2. 强杀后打开 App
  3. 出现蓝色悬浮球「魔」
  4. 点开面板：
       - 打开「强制会员」或点「立即应用强制会员」
       - 打开「去广告」
       - 「扫描会员状态」看本地识别结果

【面板说明】
  显示悬浮球     隐藏/显示球
  强制会员       写入本地 vip 标记 + Hook 常见 isVip 方法
  去广告         定时清理广告视图 + Hook 广告 load/show
  启动自动应用   下次打开 App 按开关自动执行
  扫描/明细      看识别到了哪些 key 和方法
